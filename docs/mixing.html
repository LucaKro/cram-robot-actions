Implementation Overview

Implementation

Utilizing the <strong>CRAM</strong>, written in Lisp, provides significant benefits for robot programming. The framework is designed to adapt the robot's actions based on specific task requirements and real-time sensor data.

Code Snippet


(perform 
   (an action
         (type mixing)
         (context :mix-circle) 
         (reso 12)
         (rounds 1)
         (arm (:right))
         (object ?object-container)
         (grasp :top)
         (source ?object-desig-source))
)

Summary

In summary, the code snippet directs the robot to carry out a mixing action in a circular pattern. The action is set to have a resolution of 12 and will complete one full circle.
